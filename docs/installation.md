# Installation

## Prerequisites

- **Node.js**: Version 20.0.0 or higher.

## Install via NPM

To install globally:

```bash
npm install -g @night-slayer18/leetcode-cli
```

## Verify Installation

```bash
leetcode --version
leetcode --help
leetcode
```

`leetcode` (with no arguments) launches the interactive TUI when run in an interactive terminal.
